# Переименовать файл

Команда **Переименовать файл** переименовывает указанный файл или директорию.

%html.rename-file%

**Путь**  
Укажите полный или относительный путь к директории или файлу, который нужно переименовать.

**Новое имя**  
Укажите новое имя файла или директории.
