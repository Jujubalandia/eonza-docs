# Возвратиться

Команда **Возвратиться** возвращает управление из текущего скрипта в родительский скрипт.

%html.return%

**Результирующая переменная**  
Вы можете указать имя результирующей переменной, которая будет возвращена в родительский скрипт. В этом случае, рекомендуется получать имя результирущей переменной в параметрах скрипта. В противном случае, вы можете нарушить работу родительского скрипта, если измените там значение переменной с таким же именем.

**Значение**  
Значение для результирующей переменной.
